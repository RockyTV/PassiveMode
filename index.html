Passive Mode

Passive Mode is a plugin for the Just Cause 2 Multiplayer modification.

It allows players to type in a chat command and become passive - they wont receive damage from other players, and the other players wont receive their damage.

How it works

When you first type /passive or /afk, you will become a passive player. Typing those commands again will disable the passive mode for you.

Installation

To install Passive Mode on your server, download the ZIP file provided above, and extract the folder PassiveMode to Just Cause 2 Multiplayer server location\scripts.
